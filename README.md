TAREA

Paso 1: Configuración y planificación
Configure su repositorio Git (consulte proyectos anteriores si necesita una actualización).
Configure sus archivos HTML y CSS con contenido ficticio, solo para asegurarse de tener todo vinculado correctamente.
Descargue una copia en resolución completa del archivo de diseño del proyecto y tenga una idea general de cómo necesitará exponer las cosas en su documento HTML.

Paso 2: Diseño
Comience escribiendo los elementos HTML para la barra lateral, el encabezado y los contenedores de contenido principal.
En su archivo CSS, aplique las propiedades de Grid hasta que haya creado este diseño básico.

Paso 3: Anidación
Tomándolo una sección a la vez, comience a anidar elementos secundarios debajo de los elementos principales en el HTML. Recuerda que puedes seguir fabricando contenedores de rejilla dentro de contenedores de rejilla.
En la barra lateral, utilice más cuadrículas para diseñar las secciones de navegación y marca.
En el encabezado, utilice más cuadrículas para diseñar la barra de búsqueda, la información del usuario y los botones.
Para el contenido principal, utilice más cuadrículas para diseñar los proyectos, anuncios y temas de tendencia.
Complete algunos contenidos ficticios e imágenes de marcadores de posición para que pueda posicionar todos los elementos de su cuadrícula.

Paso 4: Recopilar activos
Una vez que haya completado el diseño de la cuadrícula, puede recrear el ejemplo del panel de control anterior o diseñar su propio diseño.
Todos los íconos y más se pueden descargar como SVG desde Iconos de diseño de materiales.
¡Elige tus propias fuentes! El ejemplo de diseño utiliza Roboto, que está disponible con fuentes de Google. (Quizás quieras revisar el Más lecciones sobre estilos de texto para obtener información sobre cómo incluir fuentes externas en sus proyectos).

Paso 5: ¡Algunos consejos!
Al crear el diseño, aplique colores de fondo o bordes a sus contenedores para ayudarlo a visualizar su cuadrícula.
Depende de usted si desea utilizar píxeles, fr unidades o ambas para sus pistas de cuadrícula.
Este proyecto no tiene que ser responsivo, pero si lo desea, puede expandir o reducir la sección de tarjetas de proyecto al cambiar el tamaño de la ventana del navegador.
No es necesario hacer una coincidencia perfecta de píxeles con el ejemplo de diseño. Considere esta una oportunidad para practicar sus habilidades CSS con sus propios diseños.
No olvides enviar tu panel terminado a GitHub. ¡Utilice GitHub Pages para publicarlo en todo el mundo!
